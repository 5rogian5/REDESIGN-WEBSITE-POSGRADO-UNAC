Prueba de Header con Dropdown

Esta página es para probar específicamente la funcionalidad del dropdown del header.

Coloca el mouse sobre las opciones del menú "LA ESCUELA", "ADMISIÓN" y "CONÓCENOS" para ver los dropdowns.

Instrucciones de prueba:

  1. Coloca el mouse sobre "LA ESCUELA" - debería aparecer un dropdown con opciones como Presentación, Misión y Visión, etc.
  2. Coloca el mouse sobre "ADMISIÓN" - debería aparecer un dropdown con opciones como Proceso de Admisión, Cronograma Académico, etc.
  3. Coloca el mouse sobre "CONÓCENOS" - debería aparecer un dropdown con opciones como Transparencia, Calidad, etc.
Estado esperado: Los dropdowns deberían aparecer suavemente al hacer hover y desaparecer cuando quitas el mouse.
